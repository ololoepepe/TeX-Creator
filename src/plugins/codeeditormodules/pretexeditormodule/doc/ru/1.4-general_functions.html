Функции общего назначения


\isEmpty{value}
Возвращает 1, если значение value является пустым (не содержит никаких данных). В противном случае возвращает 0.

\toInteger{value}
Преобразует значение value в целое число и возвращает это число.

\toReal{value}
Преобразует значение value в действительное число и возвращает это число.

\toString{value}
Преобразует значение value в строку и возвращает эту строку.

\format{value}{fmt}
Преобразует значение value в строку формата fmt и возвращает эту строку. Если формат недопустим или тип значения не соответствует формату, вызывается ошибка. Допустимые форматы:
[n.]i[.m] - целое число с m нулями после десятичной точки (опционально). Общая длина равна n (опционально). Недостающие символы в начале строки заполняются нулями.
[n.]<f>[.m] - действительное число с m нулями после десятичной точки (опционально). Общая длина равна n (опционально). Недостающие символы в начале строки заполняются нулями. <f> - формат действительного числа, принимает следующие значения: f - "[-]9.9", e - "[-]9.9e[+|-]999", E - "[-]9.9E[+|-]999", g - выбирается вариант e или f, в зависимости от того, какой больше подходит, G - выбирается вариант E или f, в зависимости от того, какой больше подходит.
b - "false" или "true", в зависимости от того, равняется ли value нулю или нет.
При несоответствии параметра fmt допустимым значениям вызывается ошибка.

\if{cond}{body1}[body2]
Если условие cond равняется 1, выполняется тело body1, в противном случае выполняется тело body2 (если оно имеется). Если cond не является целым числом, вызывается ошибка.

\wait{number}[fmt]
Программа ожидает number единиц времени (по умолчанию - секунды). Если задан параметр fmt, единицы времени меняются. Допустимые значения fmt:
ms, msec, millisec, millisecond, milliseconds - миллисекунды.
s, sec, second, seconds - секунды.
m, min, minute, minutes - минуты.
При несоответствии параметра fmt допустимым значениям вызывается ошибка.

\for{id}{lb}{ub}{step}{body}
Объявляет локальную переменную с идентификатором id, присваимает ей начальное значение lb и выполняет тело цикла body до тех пор, пока значение переменной id не станет раным (или не превысит) значение ub. После каждого выполнения тела цикла значение переменной id увеличивается на значение параметра step. В качестве значений lb, ub и step допустимы целые или действительные числа.

\while{cond}{body}
Пока условие cond не равно нулю, выполняет тело цикла body. Условие вычисляется перед каждым выполнением тела цикла. Оно должно иметь целочисленный тип. В противном случае вызывается ошибка.

\doWhile{body}{cond}
Выполняет тело цикла body, пока условие cond не равно нулю. Условие вычисляется после каждого выполнения тела цикла. Оно должно иметь целочисленный тип. В противном случае вызывается ошибка.

\until{cond}{body}
Пока условие cond равно нулю, выполняет тело цикла body. Условие вычисляется перед каждым выполнением тела цикла. Оно должно иметь целочисленный тип. В противном случае вызывается ошибка.

\doUntil{body}{cond}
Выполняет тело цикла body, пока условие cond равно нулю. Условие вычисляется после каждого выполнения тела цикла. Оно должно иметь целочисленный тип. В противном случае вызывается ошибка.

\return{value}
Прерывает выполнение пользовательской функции. Функция вернет значение value. Использование \return вне тела пользовательской функции вызывает ошибку.

\break{}
Прерывает выполнение цикла. Использование вне тела цикла вызывает ошибку.

\continue{}
При вызове этой функции цикл переходит на следующую итерацию. Использование вне тела цикла вызывает ошибку.
