Логические функции


\not{value}
\!{value}
Принимает значение целочисленного типа, в противном случае вызывается ошибка. Если значение value равно 0, возвращает 1, в противном случае возвращает 0.

\equal{value1}{value2}[valueN][...]
\=={value1}{value2}[valueN][...]
Принимает от двух до бесконечности значений любого типа. Если все значения равны, возвращает 1, в противном случае (если хотя бы одна пара значений не равна) возвращает 0. Если все значения являются целыми числами, сравнение производится по правилам сравнения целых чисел. Если хотя бы одно значения является действительным числом, все значения приводятся к типу действительных чисел. Если хотя бы одно значение является строкой, все значения приводятся к строковму типу.

\notEqual{value1}{value2}
\!={value1}{value2}
Аналогична \equal, однако сравнивает значения на неравенство. Если хотя бы одна пара значений не равна, возвращает 1, в противном случае (если все значения равны) возвращает 0.

\lesser{value1}{value2}
\<{value1}{value2}
Сравнивает значения value1 и value2. Если первое меньше второго, возвращает 1, в противном случае возвращает 0. Приведение типов производится так же, как в функции \equal.

\lesserOrEqual{value1}{value2}
\<={value1}{value2}
Сравнивает значения value1 и value2. Если первое меньше либо равно второму, возвращает 1, в противном случае возвращает 0. Приведение типов производится так же, как в функции \equal.

\greater{value1}{value2}
>{value1}{value2}
Сравнивает значения value1 и value2. Если первое больше второго, возвращает 1, в противном случае возвращает 0. Приведение типов производится так же, как в функции \equal.

\greaterOrEqual{value1}{value2}
>={value1}{value2}
Сравнивает значения value1 и value2. Если первое больше либо равно второму, возвращает 1, в противном случае возвращает 0. Приведение типов производится так же, как в функции \equal.

\or{value1}{value2}[valueN][...]
\||{value1}{value2}[valueN][...]
Сравнивает значения value1, value2 и так далее с единицей. Если хотя бы одно из значений равно единице, возвращает 1. Если ни одно значение не равно единице, возвращает 0. Значения должны быть целыми числами. В противном случае вызывается ошибка.

\and{value1}{value2}[valueN][...]
\&&{value1}{value2}[valueN][...]
Сравнивает значения value1, value2 и так далее с нулем. Если хотя бы одно из значений равно нулю, возвращает 0. Если ни одно значение не равно нулю, возвращает 1. Значения должны быть целыми числами. В противном случае вызывается ошибка.

\xor{value1}{value2}[valueN][...]
Выполняет операцию исключающее "или" над значениями value1, value2 и так далее. Возвращает результат операции (0 или 1). Значения должны быть целыми числами. В противном случае вызывается ошибка.
