Редактор


Основное

Текстовый радактор TeX Creator очень похож на любой другой текстовый редактор. Он предоставляет все основные возможности редактирования текста, такие как копирование, вставка, отмена и возврат предыдущего действия и т.д.


Меню Файл
Создать
Создает новый пустой документ с расширением .tex
Открыть...
Вызывает диалог открытия файла.
Открыть заново
Открывает текущий файл заново с использованием указанной кодировки.
Недавние файлы
Повторно открывает один из недавно открывавшихся файлов.
Сохранить
Сохраняет текущий документ. Если документ не существует на диске, открывается диалог сохранения файла.
Сохранить как...
Вызывает диалог сохранения файла, с помощью которого можно сохранить документ в желаемой папке.
Сохранить все
Сохраняет все документы, в которых имеются несохраненные изменения. Аналогично выполнению действия Сохранить для каждого измененного документа.
Закрыть
Закрывает текущий документ.
Закрыть все
Закрывает все открытые документы.
Выход
Закрывает окно редактора.

Меню Правка
Отменить
Отменяет последнее действие: ввод символа, вставку и т.д.
Вернуть
Возвращает последнее отмененное действие.
Вырезать
Удаляет выделенный текст из текущего документа и помещает его в буфер обмена.
Копировать
Копирует выделенный текст из текущего документа в буфер обмена.
Вставить
Вставляет текст из буфера обмена в текущий документ.
Найти...
Открывает диалог поиска/замены текста.
Найти следующее
Аналогично нажатию кнопки Найти в диалоге поиска/замены.
Заменить следующее
Аналогично нажатию кнопки Заменить в диалоге поиска/замены.
Настройки
Открывает диалог настроек программы.

Меню Документ
Сделать основным
Показывает, что текущий документ не является основным. Помечает текущий документ как основной.
Убрать основной
Показывает, что текущий документ является основным. Снимает с документа метку "основной".
Сделать закладку
Добавляет закладку для документа на текущей позиции курсора.
Следующая закладка
Переходит к следующей закладке в текущем документе.
Режим: строки
Показывает, что редактор находится в режиме работы со строками. Переключает редактор в режим блоков.
Режим: блоки
Показывает, что редактор находится в режиме работы с прямоугольными блоками. Переключает редактор в режим строк.
Переключить раскладку
Меняет выделенный текст так, как если бы он был введен при другой раскладке клавиатуры.

Очищает содержимое консоли.
Скомпилировать

Скомпилировать и открыть
Запускает текущий документ на компиляцию, после чего открывает полученный pdf-файл.
Показать
Открывает pdf-файл, соответствубщий текущему документу.

Содержание
Открывает файл справки, перечисляющий все доступные разделы справки.
Конеткстная справка
Открывает файл справки, соответствующий активному в данный момент элементу приложения.

Открывает диалог с информацией о программе: список разработчиков, лицензия и т.д.

Прочее

В режиме строк возможно изменение регистра букв в выделенном фрагменте текста. Для этого достаточно выделить нужный фрагмент и нажать Alt+Стрелка вверх или Alt+Стрелка вниз, чтобы сделать выделенные буквы заглавными или строчными соответственно.

Также в режиме строк имеется возможность двигать выделенный текст влево и вправо. Для этого используются сочетания клавиш Alt+Стрелка влево и Alt+Стрелка вправо соответственно. При этом текст будет урезаться на границе выделения. При сдвиге текста вправо урезается только последняя строка (в том числе, если она единственная). При сдивге текста влево все строки урезаются по левому краю.

Переключаться между открытыми документами можно при помощи сочетания клавиш <Ctrl+Tab>
