Консоль


Основное

В консоли отображается вывод компилятора (в том числе сообщения об ошибках). Для копмиляции необходимо, чтобы в системе был установлен один из компиляторов: pdflatex, pdftex, latex или tex. Рекомендуется использовать следующую реализацию утилиты: MiKTeX.

Выбрать соответствующий компилятор можно в выпадающем списке.

В поле "Параметры компилятора" можно задать дополнительные параметры компилятора. Чтобы посмотреть полный список доступных параметров, откройте терминал (консоль) и выполните команду pdftex -help (вместо pdftex укажите нужный компилятор).

Параметры -interaction и -output-directory уже заданы программно, поведение программы при их повторном задании не определено, поэтому настоятельно рекомендуется не указывать данные параметры.


Действия
Очистить консоль
Очищает окно вывода компилятора.
Собрать текущий документ
Вызывает утилиту, компилирующую текущий файл. Вывод компилятора отображается в соответствующем окне.
Собрать и запустить текущий документ
Вызывает утилиту, компилирующую текущий файл. Вывод компилятора отображается в соответствующем окне. Если компиляция прошла успешно, QATE попытается открыть полученный файл в программе по умолчанию, если такая программа установлена в системе.
